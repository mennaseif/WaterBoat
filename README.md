WaterBoat 🚤

A simple frontend landing page for a water boat theme, built with HTML & CSS.
This project is created for practicing frontend design, layout styling, and responsive UI.

🌊 Features

Clean and modern landing page design

Responsive layout (works on desktop and mobile)

Styled with CSS usong bootstrap5

Organized project structure with images and styles separated

📂 Project Structure
WaterBoat/
│── index.html      # Main HTML file
│── index.css       # Stylesheet
│── images/         # Project images

🚀 How to Run Locally

Clone the repository:

git clone https://github.com/mennaseif/WaterBoat.git


Navigate to the project folder:

cd WaterBoat


Open index.html in your browser.
